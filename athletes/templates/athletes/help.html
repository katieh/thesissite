How to Download GPX Files

Garmin

  1. Go to Garmin Connect
  2. Navigate to the activity whose GPX file you'd like to download.
  3. Click on the button in the top right corner.
  4. Click on "Export to GPX" from the dropdown menu. The downloaded file should end in ".gpx" (e.g. activity_1654467998.gpx)

Polar

  1. Go to Polar Flow
  2. Navigate to the activity whose GPX file you'd like to download.
  3. Scroll to the bottom of the page and click on "Export Session" (bottom left)
  4. Click on "Route (GPX)" from the pop-up window menu. The downloaded file should end in ".gpx" (e.g. Katie_Hanss_2017-03-29_16-58-05.gpx)

How to Download FIT Files

Garmin

  1. Go to Garmin Connect
  2. Navigate to the activity whose FIT file you'd like to download.
  3. Click on the button in the top right corner.
  4. Click on "Export Original" from the dropdown menu. The downloaded file should end in ".zip" (e.g. 19204901.zip)
  5. Double click on the ".zip" file and it will unzip into a ".fit" file
