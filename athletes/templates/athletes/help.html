How to Download Your Fit Files

Garmin

  1. Go to Garmin Connect
  2. Navigate to the activity whose FIT file you'd like to download.
  3. Click on the button in the top right corner.
  4. Click on "Export Original" from the dropdown menu. The downloaded file should end in ".zip" (e.g. 19204901.zip)
  5. Double click on the ".zip" file and it will unzip into a ".fit" file
